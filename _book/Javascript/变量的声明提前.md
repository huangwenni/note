### 变量的声明提前
使用var关键字声明的变量，会在所有的代码执行之前被声明（但不会提前赋值）。

但是如果声明变量时不使用var关键字，则变量不会被声明提前。
```
console.log(a);
var a = 123;
//undefined
```
### 函数的声明提前
使用函数声明形式创建的函数 function 函数（）{}。

它会在所有代码执行之前被创建，所以可以在函数声明前来调用函数。

使用函数表达式创建的函数，不会被提前创建。
```
fun();
//函数声明，会被提前创建
function fun() {
    console.log("fun函数");
}
//"fun函数"
```
```
fun2();
//函数表达式，不会被提前创建
var fun2 = function() {
    console.log("fun2函数");
};
//提前声明，但没有提前赋值，会报错。
```
