vertical-align垂直对齐

vertical-align不影响块级元素中的内容对齐，它只针对于行内元素或者行内块元素，特别是行内块元素。

通常用来控制图片/表单与文字的对齐。

默认是文字和图片的基线对齐。

baseline 默认
