#### 选择器和CSS样式层叠
##### 后代选择器
选择所有后代的意思。
```
ul li a{
	color:red;
}
```
##### 子代选择器
亲儿子。
```
ul li >a{
	color:red;
}
```
##### 交集选择器
不常用，不建议使用。
```
div.blue{
	color:blue;
}
<div class=”blue”></div>
```
##### 链接伪类选择器
:link	未访问的链接

:visited	 已访问的链接

:hover 鼠标经过链接

:active 选定的链接（鼠标按下时的样子）

有顺序要求，必须按顺序写，否则无效。
### css样式层叠
#### 层叠性
1、样式冲突，遵循的原则是就近原则，哪个样式离结构近，就执行哪个样式。

2、样式不冲突，不会层叠。

#### 优先级
!important>行内样式>ID选择器>类选择器>标签选择器>*
```
div{
	color:hotpink!important;
}
```
```
div p{ }   
p{ }
会显示上面的，因为权重会叠加，0001+0001=0002
```
**注意**

继承的权重是0。
