Vue的安装

Vue的安装

安装node.js

查询是否安装成功：

node –v

npm –v

全局安装脚手架工具vue-cli

npm install --global vue-cli

安装webpack

npm install -g webpack

初始化vue项目（Use ESLint to line your code ?选择 No ）

vue init webpack myVue

进入到myVue目录下，使用npm install安装

cd myVue npm install

运行项目

npm run dev

进入vue初始页面

http:// localhost:8080

结束项目运行

ctrl+v，选择Y
